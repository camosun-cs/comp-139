Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • Reading JavaDoc APIs
  • Interfaces
  • Exceptions
  • Polymorphism

Task 1: Using Interfaces

  1. Create an interface to describe the common functionality of the following three classes, then modify the classes to use it.

    Include the modified classes in your lab package to submit with the rest of your work in this assignment.

    class Bicycle Download Bicycle.java
    class Horse Download Horse.java
    class Freezer Download Freezer.java
  2. Create at least one more class that implements your new interface.

  3. Write a main method class named Lab2Task1 that demonstrates how you can use your interface and polymorphism techniques to do some useful work with all of your implementations.

Task 2: Using API Documentation

  1. Create the class Rational as described by API documentation given below.

    Make sure that you read and understand the API documentation for the BigInteger, and BigDecimal classes; implementing Rational will be a lot easier if you use the methods provided by these classes effectively!

  2. Write a main method class named Lab2Task2 that demonstrates creating and using a Rational instance.

Submission

Completing all tasks in this lab should result in at least 7 separate Java files within a single package named like LastnameFirstname_lab2. Compress the package directory into a ZIP archive and upload it to the D2L assignment.

The marks for this lab are heavily weighted towards good coding practice and style. Pay attention to your code formatting, make sure you have documented all public members using JavaDoc, and make sure you are using constants, arrays, loops, and methods effectively.

NOTE: This assignment is to be done individually. You can help one another with problems and questions, but in the end everyone must do their own work.
