Interface TreeNode<E>

Type Parameters:
E - type of stored element
public interface TreeNode<E>
A node in a binary tree. Each node stores a single element and links to left, right, and parent nodes.

Note that TreeNodes have a getParent() method, but no corresponding setParent. The parent of a node should only ever be changed indirectly by invoking the parent node's setLeft(TreeNode) or setRight(TreeNode) method.
